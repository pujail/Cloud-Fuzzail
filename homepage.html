Sejarah Singkat Cloud Computing

Konsep "cloud computing" berakar pada tahun 1960-an melalui J.C.R. Licklider dan konsepnya tentang "Galactic Network", di mana ia membayangkan komputer yang terhubung secara global tempat setiap orang dapat mengakses data dan program.

Era Modern Dimulai

Namun, cloud computing seperti yang kita kenal sekarang baru benar-benar muncul pada tahun 1999 dengan berdirinya Salesforce.com. Mereka memelopori konsep pengiriman aplikasi perusahaan melalui antarmuka web sederhana, membuktikan bahwa perangkat lunak dapat dikirimkan sebagai layanan (Software-as-a-Service atau SaaS).

Kelahiran Infrastruktur Cloud

Lompatan besar berikutnya terjadi pada tahun 2006 ketika Amazon Web Services (AWS) diluncurkan. AWS memulai dengan menawarkan layanan infrastruktur (Infrastructure-as-a-Service atau IaaS) seperti Elastic Compute Cloud (EC2) dan Simple Storage Service (S3). Ini memungkinkan developer untuk menyewa kapasitas komputasi dan penyimpanan sesuai kebutuhan tanpa harus membeli perangkat keras fisik.

Sejak saat itu, raksasa teknologi lain seperti Google (dengan Google Cloud Platform) dan Microsoft (dengan Azure) mengikuti, menciptakan industri multi-miliar dolar yang menjadi tulang punggung internet modern.
